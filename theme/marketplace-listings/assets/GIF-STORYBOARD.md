## Recording Instructions

### Pre-Recording Setup

1. **Browser**: Chrome with clean profile (no extensions visible)
2. **Resolution**: 1600x1000 window (will scale down)
3. **Zoom**: 100%
4. **Theme**: Light mode
5. **Content**: Use demo PDF with clear title

### Recording Steps

1. Set up screen recording at 800x500 region
2. Start recording
3. Follow storyboard scenes in order
4. Keep mouse movements smooth and deliberate
5. Pause briefly (0.5s) on key states
6. Stop recording
7. Trim and optimize

### Post-Processing

```bash
# Optimize with gifsicle
gifsicle -O3 --colors 128 input.gif -o output.gif

# Further compress if needed
gifsicle --lossy=80 output.gif -o final.gif
```

---

## Alternative: Video + Convert

If recording as video first:

```bash
# Convert MP4 to GIF using ffmpeg
ffmpeg -i demo.mp4 -vf "fps=15,scale=800:-1:flags=lanczos" -c:v gif demo.gif

# Or use gifski for better quality
gifski --fps 15 --width 800 -o demo.gif demo.mp4
```
