Marketing website theme for PDF Embed & SEO Optimize plugin.

== Description ==

A modern, accessible, and SEO-optimized WordPress theme for the PDF Embed & SEO Optimize plugin marketing website.

= Features =

* **Block Editor Ready** - Full support for the WordPress block editor with custom patterns
* **Accessible** - WCAG 2.1 AA compliant with proper ARIA labels, skip links, and keyboard navigation
* **SEO Optimized** - Compatible with Yoast SEO for meta tags, Open Graph, and Twitter Cards
* **Responsive Design** - Mobile-first approach with optimized layouts for all viewports
* **Performance Optimized** - Self-hosted fonts with font-display: swap, lazy loading, and minimal JS
* **Picture Element Support** - All images use <picture> with WebP and fallback for optimal loading
* **Custom Block Patterns** - Pre-built patterns for Hero, Features, CTA, FAQ, Pricing, and Comparison sections

= Block Patterns =

The theme includes the following block patterns (found in Patterns > PDF Viewer Theme):

1. **Hero Section** - Main landing page hero with CTAs
2. **Features Grid** - Feature cards with icons
3. **Call to Action** - CTA section with buttons
4. **FAQ Accordion** - Expandable FAQ section
5. **Comparison Table** - Feature comparison table
6. **Pricing Cards** - Pricing tiers display

= Requirements =

* WordPress 6.0 or higher
* PHP 8.0 or higher
* Recommended: Yoast SEO plugin for meta tags

== Installation ==

1. Upload the `pdfviewer-theme` folder to the `/wp-content/themes/` directory
2. Activate the theme through the 'Appearance' > 'Themes' menu in WordPress
3. Download and install the font files (see Font Installation below)
4. Configure the theme options in 'Appearance' > 'Customize'
5. Set up navigation menus in 'Appearance' > 'Menus'
6. Install and configure Yoast SEO for meta tags

= Font Installation =

The theme uses self-hosted Inter and Outfit fonts. Download them from:

**Inter Font:**
- https://fonts.google.com/specimen/Inter
- Extract and copy to: /assets/fonts/inter/
- Required files: inter-regular.woff2, inter-medium.woff2, inter-semibold.woff2

**Outfit Font:**
- https://fonts.google.com/specimen/Outfit
- Extract and copy to: /assets/fonts/outfit/
- Required files: outfit-regular.woff2, outfit-medium.woff2, outfit-semibold.woff2, outfit-bold.woff2, outfit-extrabold.woff2

= Menu Locations =

1. **Primary Menu** - Main navigation in header
2. **Footer Menu** - Footer resources links
3. **Legal Links** - Footer legal links (Imprint, Privacy, Contact)

== Customization ==

= Theme Options =

Navigate to 'Appearance' > 'Customize' to configure:

* **Theme Options** - WordPress and Drupal download URLs
* **Social Links** - Legal pages and company website URLs

= Editing Pages =

All pages are fully editable using the WordPress block editor. Use the included block patterns to quickly build pages:

1. Edit a page in the block editor
2. Click the "+" button to add a block
3. Go to "Patterns" tab
4. Select "PDF Viewer Theme" category
5. Insert desired patterns

== Accessibility ==

The theme follows WCAG 2.1 AA guidelines:

* Skip link for keyboard navigation
* Proper heading hierarchy
* ARIA labels on interactive elements
* Sufficient color contrast
* Focus visible states
* Reduced motion support

== Performance ==

* Self-hosted fonts with font-display: swap
* Native lazy loading for images
* Minimal JavaScript (vanilla JS, no libraries)
* Optimized CSS with only used classes
* WebP images with fallbacks
