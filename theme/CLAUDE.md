# CLAUDE.md - PDF Embed & SEO Optimize Website

## Project Overview

This is a **marketing website** for the "PDF Embed & SEO Optimize" WordPress/Drupal plugin. Built with **Vite + React + TypeScript**, it showcases the plugin's features, provides documentation, and handles e-commerce (Pro tier).

**Live Site:** https://pdfviewer2026.drossmedia.de/
**Target WordPress Site:** https://pdfviewer.drossmedia.de/

---

## WordPress Theme Download

The converted WordPress theme is available in the `wp-theme/pdfviewer-theme/` directory.

**Download Links:**

| Method | URL |
|--------|-----|
| **Download Theme ZIP (Direct)** | https://github.com/alexanderdross/PDFviewer-Website/raw/main/downloads/pdfviewer-theme.zip |
| **Browse Theme Files** | https://github.com/alexanderdross/PDFviewer-Website/tree/main/wp-theme/pdfviewer-theme |
| **Download Full Repository (ZIP)** | https://github.com/alexanderdross/PDFviewer-Website/archive/refs/heads/main.zip |

**Installation:**
1. Download the theme using one of the methods above
2. If downloading the full repository, extract and locate `wp-theme/pdfviewer-theme/`
3. Upload the `pdfviewer-theme` folder to `wp-content/themes/` on your WordPress installation
4. Activate the theme from **Appearance > Themes** in WordPress admin

## Notes for Development

### Design Replication Requirements
- **Exact visual fidelity** to the Vite site (https://pdfviewer2026.drossmedia.de/)
- Same colors, typography, spacing, and layout
- Identical responsive breakpoints and mobile design
- Preserve all animations and hover effects
- Match all gradients, shadows, and glassmorphism effects

### Technical Requirements
- Preserve all SEO elements (critical for product)
- Keep accessibility features (ARIA, focus states, skip links)
- Ensure fast loading times (Core Web Vitals)
- Use Alpine.js for React-like interactivity (menus, tabs, accordions)
- Self-host Inter and Outfit fonts

### What NOT to Convert
- `/pdf/` pages - handled by PDF Embed plugin
- `/pdf-grid/` - handled by PDF Embed plugin
- `/pdf/example-X/` - handled by PDF Embed plugin
- The theme should style plugin output to match the overall design
